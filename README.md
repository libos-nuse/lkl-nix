# nixpkgs-lkl

## Current Packages

- qemu-circle (https://github.com/rsta2/qemu)
- nix-selinux-policy

## Getting Started

Clone this repository into `~/.config/nixpkgs/overlays`:

```sh
mkdir -p ~/.config/nixpkgs/overlays
cd ~/.config/nixpkgs/overlays
git clone https://github.com/libos-nuse/nixpkgs-lkl.git
```
